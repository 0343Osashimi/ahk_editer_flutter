# 薬歴ソフト上で AO の薬歴を自動入力する

## 実行用ファイルを起動する

**AO.自動入力用.exe** を開きます。これはAOの実行に必要な最小限のファイルです。  
起動後、薬歴ソフトのテキストボックスに**@bp1**と入力後、スペースを押してみましょう。  
血圧の薬歴が入力されたら成功です✨  

## 薬歴ショートカットについて
|  ショートカット  |  薬歴の種類  |
| ---- | ---- |
|  @bp1  |  降圧薬を利用していて、安定している方向けの薬歴  |
|  @meruka  |  メルカゾールを使用している方向けの薬歴  |
| @ua1 | 尿酸値が高めの方向けの薬歴 |  

以上の他、たくさんの薬歴ショートカットがあらかじめ準備されています。  
これらの薬歴の詳細をみるために、 **AO.薬歴編集用.exe** を使いましょう。

## AO.薬歴編集用.exeについて
**AO.薬歴編集用.exe** では、薬歴の確認・編集ができます。  
まず最初の画面で **設定ファイルを読み込む** ボタンを押しましょう。  
これでAOに薬歴データを読み込ませることが出来ました。  
  
さらに、左のサイドバーの上から3つ目のアイコンを押してみましょう。  
この画面の「現在のデータ」から現在読み込まれている薬歴の詳細を調べることが出来ます。  
  
ホットキーと書かれている **@～～** というのが、その薬歴を呼び出すためのショートカットです。  
   
  
## 自分専用の薬歴データの作成
**AO.薬歴編集用.exe** のサイドバーの上から2番目にあるペンのアイコンをクリックします。  
ここから順番に沿って薬歴を入力すると、自分専用の薬歴を編集可能です。

### 注意：薬歴データを作成したあとは、必ず「実行用ファイル作成」を行う必要があります。

## 実行用ファイル作成
薬歴データを作成した後は、それを　**AO.自動入力用.exe**　に反映する必要があります。その方法はカンタンです。  
薬歴データを書いた後、サイドバーの上から**3番目のアイコン**をクリック → **実行用ファイル作成** をクリックすると新しい　**AO.自動入力用.exe**　が作成されます。  

あたらしい **AO.自動入力用.exe**　を実行すればOKです。
　　
　　
## AOをPCの起動時に自動で実行するようにする
PC起動のたびに、いちいち　**AO.自動入力用.exe**　をクリックするのは面倒です。  
自動で実行してくれるように設定しましょう！  
  
やり方はカンタン。  
**起動時にAO自動起動を設定するためのファイル.exe** をクリック → **管理者として実行** をクリックするだけ。  

これで、PC起動時にAOを実行するように設定できました。  

もっと詳しく操作方法を見たい方は、次の章からのマニュアルをご覧ください。
