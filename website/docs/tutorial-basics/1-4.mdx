# 4. 薬歴ソフト上で AO の薬歴を自動入力する

## 4-1. 実行用ファイルを起動する

<img src={useBaseUrl("/img/output1.webp")} />

AO のフォルダの中に**output**フォルダがあります。ここに実行用ファイルが保存されています。

<img src={useBaseUrl("/img/output2.webp")} />

**output.exe**が実行用ファイルの本体なので、これを起動します。

<img src={useBaseUrl("/img/output3.webp")} />

タスクバーにこのアイコンが表示されたら成功です。

## 4-2. 薬歴を自動入力する

この状態で薬歴ソフト上で薬歴データ作成時に設定した**ショートカットキー**を入力すると、薬歴データが自動入力されます。

例えば、血圧の薬歴を**@bp1**として保存している場合、薬歴ソフト上で**@bp1**を入力し、`スペース`または `Enter` を押下すると血圧の薬歴が自動入力されます。

ちなみに、アイコンを右クリックすると実行用ファイルを操作できます。**exit**で実行用ファイルを終了したり、**Suspend Hotkeys**で一時的に無効化できます。
