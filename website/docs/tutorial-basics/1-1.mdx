# 1. 設定ファイルの読み込み

AO に事前に作成した薬歴を読み込むことができます。  
AO には最初から **@bp1** という血圧の薬歴が作成されているため、何も保存していなくても **血圧の薬歴** を読み込むことができます。  
まずは読み込みを試してみましょう。

## 設定ファイルを読み込む方法

<img src={useBaseUrl("/img/ao_load.webp")} />

**設定ファイルを読み込む**ボタンをクリックするだけです！カンタンですね。  
この操作によって、AO の**setting**フォルダに保存されている薬歴データがすべて読み込まれます。
