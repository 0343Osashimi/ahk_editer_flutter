# 3. 実行用ファイルを作成する

<img src={useBaseUrl("/img/execute1.webp")} />

**実行用ファイルを作成する**ボタンをクリックすると、薬歴ソフト上で薬歴を呼び出すための**実行用ファイル**を作成することができます。

## 英語のダイアログが出てくる！

<img src={useBaseUrl("/img/execute2.webp")} />

この英語のダイアログが出てきますが、問題ありません。**OK**をクリックすると正常に実行用ファイルを生成することができます。

## 英語のダイアログが出てくる！ part2

<img src={useBaseUrl("/img/execute3.webp")} />

すでに実行用ファイルを実行している場合に、新しい実行用ファイルを作成しようとすると、このダイアログが出てきます。{" "}

この場合は**& Reload**をクリックしてください。  
Reload をクリックすると、古い実行ファイルを終了 → 新しい実行用ファイルを作成 → 新しい実行用ファイルを実行という一連の処理を自動で行います。

## ここまできたら、後は呼び出すだけです！

つぎが最後のステップです！
