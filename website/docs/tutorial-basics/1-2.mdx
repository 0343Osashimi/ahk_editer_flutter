# 2. 薬歴データの作成・保存

<img src={useBaseUrl("/img/soap1.webp")} />

この画面では、SOAP を自由に編集することができます。

<div class="mb-4 rounded-lg border border-gray-200 bg-white p-8 text-sm shadow-md">
  <p>・病名に合わせた薬歴を入力する</p>
  <p>・薬に合わせた薬歴を入力する</p>
  <p>・症状に合わせた薬歴を入力する</p>
</div>

さまざまなバリエーションを作成することが可能です。  
店舗の内情に合わせた組み合わせで薬歴データを作成してみてください。

## 2-1. 呼び出し用データの編集

<img src={useBaseUrl("/img/soap2.webp")} />

|                        | 説明                                                                                                                                                                                                            |
| ---------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 自動入力するときのキー | 薬歴ソフト上で、この薬歴を<u>呼び出すとき</u>に使うショートカットキーを設定します。**@ほにゃらら + 数字**と設定すると、覚えやすく呼び出しやすいため便利です。<br/>**@**が半角になっているかチェックしましょう。 |
| グループ               | AO 上で薬歴データが増えて来た時、整理するためのものです。設定しなくても保存可能です（\*非推奨）                                                                                                                 |
| ニックネーム           | 薬歴データの作成者として名前を残すことができます。設定しなくても保存可能です。                                                                                                                                  |

## 2-2. 薬歴データの保存

<img src={useBaseUrl("/img/soap3.webp")} />

**保存する**をクリックすると、薬歴データが保存されます。 ここで保存されたデータは**setting**フォルダに保存されるため、自由に共有することができます。

## Congratulations!🚀

これで薬歴の準備は完了です！  
次は、薬歴を呼び出すための**実行用ファイル**を作成しましょう！
