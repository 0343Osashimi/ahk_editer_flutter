# AOで書いた薬歴を共有する
AOで書いた薬歴は、ほかのすべてのAOユーザーと共有することが出来ます。  
したがって、AOを使うユーザーが多ければ多いほど、あなたの薬歴のバリエーションを増やすことができます。  

## 薬歴データの保存場所
**AO.薬歴編集用.exe** がある場所に、 **setting** フォルダがあります。  
settingフォルダの中にある **.json** ファイルが薬歴データの本体です。  
  
## 薬歴データをほかのユーザーと共有する方法
**.json** を、他のユーザーのAOの **setting** フォルダに移すだけで準備は完了です   
  
**AO.薬歴編集用.exe**を開き、**設定ファイルを読み込む** をクリックすることで、他のユーザーの薬歴を取り込むことが出来ます。  
  
  
さらに、**サイドバーの上から3個目のアイコン**をクリック → **実行用ファイルを作成** から、新しい**AO.自動入力用.exe**を作成することができます。
