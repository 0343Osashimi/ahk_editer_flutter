# 薬歴の入力スピードを調整する
AOの薬歴入力スピードは、最高速度には設定されていません。  
これは、様々な環境で使われることを想定しているため、一般的なスペックに合わせて調整されているためです。
しかし、ハイスペックなPCではさらに高速の入力が可能です。  
  
したがって、AOには**薬歴の入力スピードを調整できる機能**が存在しています。

## 入力スピードの調整方法
**AO.薬歴編集用.exe**を開き、設定ファイルを読み込んだあと、左アイコンのうえから3番目を開きましょう。  
こちらに存在する**速度/安定度設定**を調整したのち、実行用ファイルを作成すると、スピード調整された**AO.自動入力用.exe**が出力されます。

## スピードについて
スピードのデフォルト設定は**1.0**です。  
この数値は薬歴が安定して入力できて、スピードも実行に耐えうる数値として推奨されるものです。  
  
数値を大きくすると、速度が速くなります。しかし不要な改行が入力されたり、SOAPの打ち間違いなどが発生する確率が高まります。  
  
また、数値を小さくすると、速度は遅くなります。しかし安定して入力される可能性が高くなります。  
  

ぜひ自分の環境で最高のスピードとなるよう、調整にチャレンジしてみてください。
